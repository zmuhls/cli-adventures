Adventures v1.0

Welcome to CLI Adventures - Learn terminal commands through interactive exploration! You're in your home directory. Type help to see available commands.

~$
Command Challenge: Type 'help' to see available commands

Current Challenge

List the contents of your home directory
Instructions
Just type 'ls' and press Enter

Command Reference

  • ls - List directory contents
  • cd [dir] - Change directory
  • cd .. - Go to parent directory
  • cd ~ - Go to home directory
  • cat [file] - View file contents
  • pwd - Print working directory
  • mkdir [dir] - Create directory
  • touch [file] - Create empty file
  • rm [file] - Remove file
  • mv [src] [dst] - Move/rename file
  • cp [src] [dst] - Copy file
  • unzip [file.zip] - Extract zip file
  • clear - Clear the terminal
  • reset - Reset the game
  • help - Show help
